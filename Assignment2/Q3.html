Projects

  1. 1. Rawg Clone

This project allows users to browse and search for video games, view detailed information, and explore game genres.   2. 2. Tic Tac Toe

    A classic Tic Tac Toe game developed using Java. This project features a simple user interface, allowing two players to take turns and compete in a grid-based game.   3. 3. Hangman

    A fun and interactive Hangman game built with Java. Players guess letters to uncover a hidden word, with a limited number of incorrect guesses allowed. This project showcases basic Java concepts like string manipulation, loops, and conditional statements, while providing an engaging way to test vocabulary skills.
